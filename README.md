# 硅基工作台 — 供应链金融智能作业台原型

> 面向银行供应链金融场景的多样本业务作业台原型，覆盖从客群识别到贷后恢复的全链路闭环。

## 定位

这是一套**可交互的高保真原型**，用于演示"脱核链贷"场景下，银行如何在链主不确权的前提下，通过替代性证据识别、评估并授信三级供应商。

系统严格遵守银行内部系统边界：
- **不清算、不放款** — 不越权统一信贷系统，只做预审推单与状态回流查看
- **不发短信、不建商机** — 不越权 CRM 系统，只做策略生成与任务下发
- **不采数据、不练模型** — 不越权外数平台与模型工厂，只做规则配置与能力调用编排

**企业名称为公开可检索主体，供应链关系、授信状态与风险状态为演示化示例。**

## 设计原则

- **多样本驱动**：5 个企业样本贯穿全流程，`SampleSwitcher` 全局切换
- **组合层 / 样本层分离**：组合经营看背景，当前样本做主角
- **统一作业台语法**：每个场景页 = `StageStrip` + `SampleHero` + `SceneQuestion` + 核心作业区 + `AiJudgmentBlock`
- **AI 三段式输出**：当前判断 → 判断依据 → 建议动作
- **系统边界感**：预审推单而非审批放款、策略下发而非直接触达、规则配置而非数据采集
- **阶段感**：客群识别 → 预授信 → 补审 → 审批推送 → 在营监控 → 风险处置 → 贷后恢复

## 关键特性（v0.6.0）

### 架构越位修正
- **产品与审批**：终点为"预审通过，已推送信贷系统"，非本系统审批放款
- **授信资产池**：还款状态为只读看板，数据同步自统一信贷系统（T+1），无伪操作按钮
- **风险监控**：处置动作改为"任务下发至 CRM"，非本系统直接催收
- **贷后经营**：增收终点为"生成营销线索→CRM"或"生成拜访任务→展业 PAD"

### 数据工具
- **智能尽调工具**：AI 驱动一键生成企业尽调报告，跨模块通用（推荐流/筛选流/资产列表均可调用）
- **数据源管理**：外数平台接入状态、配额余量、连通性测试
- **数据源连通性探针**：规则配置界面实时检测底层数据可用性（🟢已接入 / 🟡配额不足 / 🔴未采购）

### 候选资产池增强
- **外勤录入流**：移动采集 → AI 解析 → 冲突检测 → 入池复核
- **命中特征列**：展示企业被识别依据（标签命中 / 图谱命中 / 模型命中）
- **跨模块一键尽调**：推荐流、筛选流、资产列表均可直接跳转至智能尽调工具

### 今日工作台
- 问候区 + 今日动作型指标（待补审/预警/恢复观察/待跟进/AI 建议）
- 任务优先级队列 + 当前重点客户卡 + AI 建议条 + 业务快捷入口

## 演示流程

1. 打开**今日工作台**，查看今日待办任务和重点客户
2. 进入**候选资产池** → 智能推荐流 / 条件筛选流 / 外勤录入流 → 查看命中特征 → 一键尽调
3. 进入**关系图谱** → 查看链路证明与节点详情
4. 切换至**授信资产池** → 转化看板观察客户在池间流转
5. 切换至**产品与审批** → 预审与推单 → 完成补审作业 → 推送信贷系统
6. 切换至**风险监控** → 模拟风险事件 → 处置任务下发至 CRM
7. 切换至**贷后经营** → 恢复经营 / 生成营销线索 → 完成闭环
8. 进入**策略与配置** → 智能尽调工具 → 数据源管理 → 规则连通性校验
