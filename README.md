# 硅基工作台 — 普惠资产 AI 作业中台

> 面向银行普惠金融场景的 AI 驱动智能作业中台原型，以"AI 作业阶段"为导航主线，覆盖从智能识别、尽调、审批到监控、经营的全链路闭环。

## 产品定位

这是一套**可交互的高保真原型**，演示"脱核链贷"场景下，银行如何在链主不确权的前提下，通过 AI 辅助的替代性证据识别、评估并授信三级供应商。

系统不是传统业务后台，而是一个**AI 作业指挥台**——每个一级菜单对应一个 AI 辅助的作业阶段，用户按阶段推进业务，系统在每个阶段提供判断、证据和建议。

### 系统边界（三不原则）

| 本系统做什么 | 本系统不做什么 | 由谁做 |
|-------------|--------------|--------|
| 预审推单、状态回流查看 | 清算、放款、审批建档 | 统一信贷系统 |
| 策略生成、任务下发 | 发短信、建商机、触达客户 | CRM / 企微 |
| 规则配置、能力调用编排 | 采集数据、训练模型 | 外数平台 / 模型工厂 |

**企业名称为公开可检索主体，供应链关系、授信状态与风险状态为演示化示例。**

**布局原则**：
- **顶部**只放全局能力（搜索/通知/用户），不承担导航
- **左侧**一级导航固定 140px，按 AI 作业阶段排列，蓝底高亮当前项
- **内容区**由二级导航侧边栏 + 主工作区组成，二级导航可折叠
- **底部**状态栏显示系统运行状态和当前案例上下文

## 界面设计原则

- **AI 作业阶段导航**：一级菜单按业务推进阶段排列（识别→尽调→审批→监控→经营），不按后台组织架构排
- **今日工作台 = 个人指挥台**：打开首页立刻知道今天先干什么、先看哪一户、下一步点哪里，不是经营报表
- **多样本驱动**：5 个企业样本贯穿全流程，`SampleSwitcher` 全局切换当前作业对象
- **AI 三段式输出**：每个作业页面的 AI 辅助区遵循「当前判断 → 判断依据 → 建议动作」结构
- **系统边界感**：预审推单而非审批放款、任务下发而非直接触达、规则配置而非数据采集
- **左侧导航层级**：一级按阶段排，"策略与配置"放最底部明确为后台维护层

## 关键特性（v0.7.2）

### AI 作业阶段导航
- 8 个一级菜单按作业阶段排列，从"今日工作台"到"策略与配置"
- 左侧固定导航取代顶部导航，顶部仅保留全局搜索/通知/用户
- 二级功能由 SceneLayout 侧边栏承载，支持折叠

### 今日工作台（个人指挥台）
- **今日总览**：问候区 + 5 项动作型指标 + 今日任务预览（跳转任务中心）+ 当前重点客户卡 + AI 建议条 + 6 宫格快捷入口 + 样本分布环形图
- **任务中心**：合并原「待办」与「风险提醒」为统一队列，按来源标签筛选（补审 / 预警 / 催收 / 审批 / 跟进 / 恢复），可深链至智能审批、智能监控、智能经营等
- **消息通知**：审批、AI 作业、预警、系统事件等演示消息流，支持未读与标为已读
- **我的关注**：星标客户、最近处理、AI 推荐关注（非全量客户表；全量在「授信资产池 → 在营资产」等模块查看）
- **AI 建议**：跨样本聚合的 AI 摘要与建议动作，按优先级与模型置信度排序

### 智能识别
- 名单导入：Excel/CSV 批量上传，导入记录与命中率
- 智能推荐流：AI 信息流 + 推荐理由 + 命中特征标签（标签命中/图谱命中/模型命中）
- 条件筛选流：多维筛选器 + 最近命中线索
- 候选资产列表：双维度置信度 + 阶段状态 + 跨模块一键尽调
- 关系图谱：簇状布局 + 链路证明 + 节点详情卡 + hover 显示边标签
- 公私联动验证：企业维度 + 法人维度交叉验证

### 智能尽调
- 外勤录入：PAD 端采集任务 → AI 提取预览 → 行内数据冲突检测 → 一键入池
- 材料采集：待采集/待核验/已归档/异常分类管理
- 证据核验：订单-物流-回款-发票四流交叉验证 + 覆盖度进度条
- 尽调报告：当前案件/样本的 AI 尽调报告生成与预览（企业识别 → 外数调用 → AI 分析 → 报告生成）
- 报告中心：已生成报告列表 + 版本追踪 + 导出(PDF/Word/HTML) + 提交记录 + 审阅状态

### 智能审批
- 产品匹配：产品方案卡 + 匹配规则
- 预审与推单：终点为"预审通过，已推送信贷系统"，非本系统审批放款
- 补审作业：三流证据验证 + 规则命中清单 + AI 审批建议
- 审批摘要：章节化结论（证据摘要/命中规则/风险提示/审批建议）

### 授信资产池（v1 过渡保留）
- 转化看板：候选→预授信→补审→在营漏斗 + 阶段分布
- 在营资产列表：经营状态 + 额度 + 风险标签
- 风险资产：风险信号汇总
- 还款状态看板：只读数据，标注 `[数据同步自统一信贷系统，T+1更新]`

### 智能监控
- 预警总览：风险摘要条 + 样本级风险详情
- 监控指标：阈值线 + 预测虚线 + 异常点标记
- 风险探针：12 个实时探针 + 触发状态 + 阈值/频率
- 处置任务：任务下发至 CRM，非本系统直接催收
- 规则效果：命中率/准确率/误报率 + 规则分布图

### 智能经营
- 经营总览：健康度分布 + 客户四态（正常/观察/风险/恢复）
- 客户分层：高/中/低价值分层
- 增收动作：终点为"生成营销线索→CRM"或"生成拜访任务→展业 PAD"
- 恢复经营：风险客户恢复任务列表 + 恢复周期/成功率
- 动作模板：催收/营销模板管理

### 策略与配置
- 标准规则：通用小微识别规则 + 数据源要求面板
- 场景规则：脱核链上/涉农/本地商圈等长尾场景搜索/新增/配置
- 产品配置 / 审批规则：参数配置
- 报告模板：尽调报告章节结构/内容模块/显示规则/文案口径/导出格式配置，支持模板列表 + 详情配置 + 实时预览
- 数据源管理：外数接入状态 + 配额余量 + 连通性探针（🟢已接入 / 🟡配额不足 / 🔴未采购）

## 技术栈

- **React 19** + **TypeScript**
- **Vite 6** 构建
- **Tailwind CSS 4** + **shadcn/ui** 组件库
- **Recharts** 数据可视化
- **Motion (Framer Motion)** 页面切换与微动画
- **Lucide React** 图标体系
- React Context 全局状态管理（多样本 + 样本级状态）

## 演示流程

1. 打开**今日工作台** → **今日总览**看指标与重点客户 → **任务中心**处理合并队列 → **消息通知**看系统事件 → **我的关注**盯星标与 AI 推荐 → **AI 建议**浏览跨模块洞察
2. 进入**智能识别** → 名单导入批量上传 / 智能推荐流浏览 AI 推荐 / 条件筛选流多维筛选 → 候选资产列表查看命中特征 → 关系图谱验证链路
3. 进入**智能尽调** → 外勤录入查看 PAD 采集任务 → 材料采集管理文件 → 证据核验四流交叉验证 → 尽调报告 AI 解析生成 → 报告中心管理/导出/提交
4. 切换至**智能审批** → 产品匹配选择方案 → 预审与推单推送信贷系统 → 补审作业三流验证 → 审批摘要
5. 切换至**授信资产池** → 转化看板查看漏斗 → 在营资产列表 → 还款状态看板（只读）
6. 切换至**智能监控** → 预警总览 → 风险探针查看触发状态 → 处置任务下发至 CRM → 规则效果分析
7. 切换至**智能经营** → 经营总览看健康度 → 客户分层 → 恢复经营跟踪 → 增收动作生成线索 → 动作模板管理
8. 进入**策略与配置** → 标准规则 / 场景规则配置 → 报告模板配置章节与显示规则 → 数据源管理查看连通性
