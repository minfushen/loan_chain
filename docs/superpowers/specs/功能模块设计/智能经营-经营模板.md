## 二、页面概述

### 2.1 页面定位

“经营模板”页面用于沉淀和复用标准化的客户经营策略，通过模板化方式统一管理营销、维护及增收动作，提升经营效率与策略落地能力。

### 2.2 建设目标

* **标准化经营策略**：沉淀优秀经营经验，形成可复用模板。
* **提升执行效率**：快速生成经营任务，减少人工设计成本。
* **支持精准营销**：结合客户分层，实现差异化经营。
* **强化过程管理**：统一模板管理与效果评估。

### 2.3 目标用户

* 客户经理
* 经营管理人员
* 分支行管理者
* 总行产品与运营团队

## 四、功能模块设计

### 4.1 模板概览区

#### 4.1.1 功能说明

以卡片形式展示经营模板的整体情况，帮助用户快速了解模板的使用与执行效果。

#### 4.1.2 展示指标

| 指标     | 说明              |
| ------ | --------------- |
| 模板总数   | 当前系统中已创建的模板数量   |
| 启用模板数  | 处于启用状态的模板数量     |
| 本月使用次数 | 当月被调用生成经营任务的次数  |
| 覆盖客户数  | 使用模板触达的客户数量     |
| 转化率    | 营销成功客户数 / 触达客户数 |

---

### 4.2 模板列表区

#### 4.2.1 功能说明

展示所有经营模板，支持查询、筛选、排序及操作管理。

#### 4.2.2 筛选条件

| 条件     | 说明                      |
| ------ | ----------------------- |
| 模板名称   | 模糊查询                    |
| 模板类型   | 营销类 / 维护类 / 增收类 / 风险关怀类 |
| 适用客户层级 | L1-L4                   |
| 适用产品   | 贷款 / 存款 / 结算 / 综合金融     |
| 状态     | 启用 / 停用                 |
| 创建机构   | 模板归属机构                  |
| 创建人    | 模板创建者                   |
| 创建时间   | 时间范围筛选                  |

#### 4.2.3 列表字段设计

| 字段     | 说明                |
| ------ | ----------------- |
| 模板名称   | 经营模板的名称           |
| 模板编号   | 系统自动生成            |
| 模板类型   | 营销类/维护类/增收类/风险关怀类 |
| 适用客户层级 | L1-L4             |
| 目标产品   | 模板涉及的金融产品         |
| 执行动作数  | 模板中包含的经营动作数量      |
| 使用次数   | 被调用次数             |
| 转化率    | 模板执行效果            |
| 状态     | 启用/停用             |
| 创建机构   | 所属机构              |
| 创建人    | 创建人               |
| 创建时间   | 创建日期              |
| 更新时间   | 最近修改时间            |

#### 4.2.4 操作功能

| 操作    | 说明          |
| ----- | ----------- |
| 查看    | 查看模板详情      |
| 编辑    | 修改模板        |
| 复制    | 基于现有模板创建新模板 |
| 启用/停用 | 控制模板是否可被使用  |
| 删除    | 删除未被使用的模板   |
| 生成任务  | 根据模板生成经营任务  |

---

### 4.3 模板详情抽屉

#### 4.3.1 功能说明

点击模板名称后，以右侧抽屉形式展示模板的详细信息。

#### 4.3.2 展示内容

| 模块   | 内容                 |
| ---- | ------------------ |
| 基本信息 | 模板名称、编号、类型、适用层级、状态 |
| 适用客户 | 行业、规模、区域等          |
| 经营目标 | 如提升授信使用率、增加存款沉淀等   |
| 执行动作 | 具体的营销或维护步骤         |
| 触达方式 | 电话、短信、企业微信、上门拜访等   |
| 执行周期 | 单次/周期性             |
| 责任人  | 客户经理或团队            |
| 效果指标 | 转化率、收入提升等          |
| 备注   | 其他补充信息             |

---

### 4.4 模板创建/编辑页面

#### 4.4.1 功能说明

支持新建或编辑经营模板，采用分步骤表单设计，提升用户体验。

#### 4.4.2 创建流程

```text
步骤1：基本信息
步骤2：适用客户
步骤3：经营目标
步骤4：执行动作
步骤5：效果指标
步骤6：确认发布
```

#### 4.4.3 字段设计

##### （1）基本信息

| 字段     | 类型  | 是否必填 | 说明                |
| ------ | --- | ---- | ----------------- |
| 模板名称   | 文本  | 是    | 不超过50字            |
| 模板类型   | 下拉  | 是    | 营销类/维护类/增收类/风险关怀类 |
| 适用客户层级 | 多选  | 是    | L1-L4             |
| 适用产品   | 多选  | 否    | 相关金融产品            |
| 执行周期   | 下拉  | 是    | 单次/周期性            |
| 状态     | 开关  | 是    | 启用/停用             |
| 模板描述   | 文本域 | 否    | 模板用途说明            |

##### （2）适用客户条件

| 条件   | 示例       |
| ---- | -------- |
| 行业   | 制造业、批发零售 |
| 企业规模 | 小型/中型    |
| 授信余额 | ≥ 100万   |
| 用信率  | ≤ 30%    |
| 风险等级 | 低风险      |

##### （3）经营目标

| 指标      | 示例    |
| ------- | ----- |
| 授信使用率提升 | ≥ 20% |
| 存款增长    | ≥ 10% |
| 产品渗透率   | ≥ 15% |
| 中间业务收入  | ≥ 5万元 |

##### （4）执行动作

| 字段   | 说明            |
| ---- | ------------- |
| 动作名称 | 如“授信额度激活营销”   |
| 动作类型 | 电话/短信/拜访/线上活动 |
| 执行顺序 | 支持拖拽排序        |
| 间隔时间 | 如3天后执行        |
| 话术模板 | 支持文本录入        |
| 附件   | 支持上传          |

##### （5）效果指标

| 指标    | 说明   |
| ----- | ---- |
| 触达客户数 | 自动统计 |
| 转化客户数 | 自动统计 |
| 转化率   | 系统计算 |
| 收入贡献  | 系统计算 |

---

### 4.5 模板效果评估入口

#### 4.5.1 功能说明

提供模板执行效果的分析入口，支持对模板价值进行评估和优化。

#### 4.5.2 分析维度

| 维度   | 说明       |
| ---- | -------- |
| 使用趋势 | 模板调用次数变化 |
| 客户覆盖 | 不同层级客户分布 |
| 转化效果 | 转化率对比    |
| 收入贡献 | 模板带来的收益  |
| 机构对比 | 不同机构使用效果 |

| 总行管理员  | 全量管理与发布 |
| 系统管理员  | 删除与权限配置 |
