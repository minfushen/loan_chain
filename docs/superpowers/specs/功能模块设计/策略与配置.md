
# 📘 《策略与配置》模块页面规格说明（PRD）

## 一、模块整体说明

### 1.1 模块定位

“策略与配置”是系统的**策略中枢与配置中心**，用于统一管理业务规则、场景策略、产品参数、审批流程、报告模板及数据源接入，支撑智能经营与风险决策的灵活配置与快速迭代。

### 1.2 目标用户

* 总行产品经理
* 风险管理人员
* 运营管理人员
* 系统管理员
* 数据分析人员

### 1.3 推荐二级菜单结构

```
策略与配置
├── 基础规则
├── 场景策略
├── 产品策略配置
├── 审批策略
├── 报告模板配置
└── 数据源配置
```

# 二、基础规则页面规格（PRD）

## 2.1 页面定位

用于配置系统级通用规则，是各业务场景策略的基础支撑，如准入规则、评分规则及指标计算逻辑。

## 2.2 页面结构

```
基础规则
├── 规则概览区
├── 规则列表区
├── 规则配置区（新增/编辑）
└── 规则版本管理区
```

## 2.3 核心字段设计

| 字段     | 类型     | 说明          |
| ------ | ------ | ----------- |
| 规则名称   | 文本     | 规则的唯一标识     |
| 规则类型   | 下拉     | 准入/评分/计算/标签 |
| 适用范围   | 下拉     | 全行/分支机构     |
| 规则表达式  | 表达式编辑器 | 支持逻辑运算      |
| 状态     | 开关     | 启用/停用       |
| 优先级    | 数值     | 决定规则执行顺序    |
| 版本号    | 自动生成   | 支持历史追溯      |
| 生效时间   | 日期     | 规则启用时间      |
| 创建人/时间 | 系统记录   | 审计使用        |

## 2.4 功能列表

* 新建规则
* 编辑规则
* 启用/停用
* 复制规则
* 版本管理与回滚
* 规则测试（模拟数据验证）
* 导入/导出配置

## 2.5 权限控制

| 角色    | 权限    |
| ----- | ----- |
| 总行管理员 | 全量管理  |
| 分支行   | 查看    |
| 系统管理员 | 发布与回滚 |

# 三、场景策略页面规格（PRD）

## 3.1 页面定位

面向具体业务场景（如营销、风控、客户恢复等）的策略配置中心，通过组合基础规则形成完整的业务决策逻辑。

## 3.2 页面结构

```
场景策略
├── 场景概览区
├── 场景策略列表
├── 策略编排区（规则组合）
└── 策略效果监测入口
```

## 3.3 核心字段设计

| 字段     | 类型   | 说明        |
| ------ | ---- | --------- |
| 场景名称   | 文本   | 如“新客授信营销” |
| 场景类型   | 下拉   | 营销/风控/挽回  |
| 关联基础规则 | 多选   | 引用基础规则    |
| 触发条件   | 表达式  | 事件或条件触发   |
| 决策逻辑   | 流程编排 | 支持可视化配置   |
| 输出动作   | 下拉   | 推送、审批、预警等 |
| 优先级    | 数值   | 多场景冲突处理   |
| 状态     | 开关   | 启用/停用     |

## 3.4 功能列表

* 场景策略创建与编辑
* 可视化流程编排（Rule Flow）
* 模拟运行与效果预览
* 策略冲突检测
* 策略版本管理
* 与经营动作联动

## 3.5 应用示例

* 高价值客户促活策略
* 流失客户挽回策略
* 授信预审批策略

# 四、产品策略配置页面规格（PRD）

## 4.1 页面定位

用于管理金融产品的策略参数及匹配规则，支持不同客户群体的差异化产品推荐与授信政策。

## 4.2 页面结构

```
产品策略配置
├── 产品列表区
├── 产品策略配置区
├── 产品匹配规则区
└── 参数版本管理区
```

## 4.3 核心字段设计

| 字段     | 类型  | 说明          |
| ------ | --- | ----------- |
| 产品名称   | 文本  | 金融产品名称      |
| 产品类型   | 下拉  | 贷款/存款/供应链金融 |
| 适用客户层级 | 多选  | L1-L4       |
| 授信额度区间 | 数值  | 最小/最大额度     |
| 利率区间   | 数值  | 年化利率        |
| 期限     | 数值  | 月/年         |
| 还款方式   | 下拉  | 等额本息等       |
| 匹配规则   | 表达式 | 客户准入条件      |
| 状态     | 开关  | 启用/停用       |

## 4.4 功能列表

* 产品新增/编辑/删除
* 参数配置与调整
* 产品匹配规则配置
* 产品复制
* 产品上下架管理
* 产品策略版本控制

# 五、审批策略页面规格（PRD）

## 5.1 页面定位

用于配置授信及业务流程中的审批规则与流程，实现自动化与人工审批的协同。

## 5.2 页面结构

```
审批策略
├── 审批流程列表
├── 流程设计器
├── 审批规则配置
└── 审批权限管理
```

## 5.3 核心字段设计

| 字段     | 类型   | 说明          |
| ------ | ---- | ----------- |
| 流程名称   | 文本   | 如“小微授信审批流程” |
| 审批类型   | 下拉   | 自动/人工/混合    |
| 触发条件   | 表达式  | 如授信额度>100万  |
| 审批节点   | 流程配置 | 多级审批        |
| 审批角色   | 下拉   | 客户经理/风控经理   |
| 审批时限   | 数值   | 单位：小时       |
| 是否自动通过 | 开关   | 满足条件自动审批    |
| 状态     | 开关   | 启用/停用       |

## 5.4 功能列表

* 可视化流程设计
* 审批节点配置
* 角色权限设置
* SLA时效管理
* 审批日志与追踪
* 与信贷系统集成

# 七、数据源配置页面规格（PRD）

## 7.1 页面定位

用于管理系统的数据接入、维护与监控，确保策略与模型的数据基础稳定可靠。

## 7.2 页面结构

```
数据源配置
├── 数据源列表区
├── 数据接入配置区
├── 字段映射区
└── 数据质量监控区
```

## 7.3 核心字段设计

| 字段     | 类型   | 说明            |
| ------ | ---- | ------------- |
| 数据源名称  | 文本   | 如“统一信贷系统”     |
| 数据源类型  | 下拉   | 数据库/API/文件    |
| 连接地址   | 文本   | URL或IP        |
| 认证方式   | 下拉   | 用户名密码/API Key |
| 更新频率   | 下拉   | 实时/T+1        |
| 状态     | 开关   | 启用/停用         |
| 最后同步时间 | 系统记录 | 数据监控          |
| 数据质量评分 | 系统计算 | 完整性与准确性       |

## 7.4 功能列表

* 新增/编辑/删除数据源
* 连接测试
* 字段映射配置
* 数据同步调度
* 数据质量监控
* 异常告警
