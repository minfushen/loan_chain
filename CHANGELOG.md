# Changelog

## v0.6.0 — 致命越位修正 + 智能尽调工具 + 外勤录入流 (2026-04-10)

### 架构级修正：系统"三观"对齐银行科技审查标准

本版本对系统在银行主干业务系统边界上的越权问题进行了全面修正，确保：
- **不清算、不放款** — 不越权统一信贷系统，只做"预审推单"和"状态回流查看"
- **不发短信、不建商机** — 不越权 CRM 系统，只做"策略生成"和"任务下发"
- **不采数据、不练模型** — 不越权外数平台和模型工厂，只做"规则配置"和"能力调用编排"

### 变更详情

#### 【产品与审批】预审与推单
- 二级菜单"审批流程"更名为 **"预审与推单"**
- 审批时间轴末节点改为 **"预审通过，已推送信贷系统"**（而非"审批通过"）
- 新增"信贷系统状态"回流展示区（已受理编号 / 等待正式建档）

#### 【授信资产池】还款状态看板
- 二级菜单"还款管理"更名为 **"还款状态看板"**
- 移除"标记已还""设置提醒"等伪操作按钮
- 新增数据来源标示条：`[数据同步自统一信贷系统，T+1更新]`
- 新增异常标红提醒：风险监控侦测到逾期但信贷系统未更新时高亮

#### 【风险监控】处置任务下发
- 二级菜单"处置动作"更名为 **"处置任务下发"**
- 新增架构边界声明条：本系统仅生成处置策略与任务，实际催收由 CRM 执行
- "执行催收"按钮改为 **"生成催收任务推送CRM"**
- 任务状态显示"已为XX经理在CRM创建催收跟进任务"

#### 【贷后经营】增收动作
- 增收动作终点改为 **"生成营销线索→CRM"** 或 **"生成外勤拜访任务→展业PAD"**
- 新增推送目标选择器（CRM / 展业PAD）
- 新增架构边界声明条

#### 【数据与接入】数据源连通性探针
- 规则配置界面每个数据源旁新增连通性探针图标
- 三种状态：🟢已接入且有余量 / 🟡已接入但配额不足 / 🔴外数平台未采购
- 🔴状态时禁用规则"保存"按钮并提示"底层平台无此数据，规则无法生效"

#### 【数据与接入】智能尽调工具（新增）
- 新增二级菜单 **"智能尽调工具"** — 通用数据工具，支持跨模块调用
- 功能：输入企业名/信用代码 → 上传补充材料 → AI 自动调用工商/税务/司法/征信外数接口 → 生成结构化尽调报告
- 流程管线可视化：企业识别 → 外数调用 → AI分析 → 报告生成
- 报告列表支持三种状态（已生成 / AI解析中 / 生成失败）及对应操作
- 可从候选资产池各流（智能推荐流 / 条件筛选流 / 候选资产列表）一键调用

#### 【数据与接入】数据源管理（新增）
- 新增二级菜单 **"数据源管理"** — 外数平台接入状态与配额管理
- 表格展示 8 个数据源的供应商、连通状态、配额、已用量、关联模块
- 支持全量连通性测试、配额扩容申请

#### 【候选资产池】命中特征显性化
- 候选资产列表新增 **"命中特征"列**，展示企业被识别的依据来源
- 格式示例：`[标签命中] 高新制造 + [图谱命中] 二级供应商 + [模型命中] 预授信通过`

#### 【候选资产池】外勤录入流（新增）
- 新增二级菜单"外勤录入流"，定位：移动采集 → AI解析 → 入池复核
- 流程管线：PAD/手机采集 → AI自动解析 → 冲突检测 → 复核入池
- 任务列表含 AI 提取结果预览、置信度评分、数据冲突告警
- 操作按钮：确认入池 / 标记冲突待核实 / 退回重采

#### 【候选资产池】跨模块一键尽调入口
- 智能推荐流每个企业卡片新增"一键尽调"按钮
- 条件筛选流每条命中线索新增"一键尽调"按钮
- 候选资产列表新增"操作"列，每行含"尽调"快捷按钮
- 点击后跳转至【数据与接入】→【智能尽调工具】

#### 【今日工作台】首页重构
- 从"经营分析首页"重构为 **"今日工作台"**
- 新增问候区、今日工作概览（动作型指标）、任务区、当前重点客户卡、AI 建议条、业务快捷入口
- 移除大型经营图表、演示进度时间轴、长解释文字
