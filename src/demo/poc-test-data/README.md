## 测试覆盖目标

这批名单设计用于验证以下主链节点是否能正常跑通：

```
文件导入 → 字段校验 → 样本生成 → 智能推荐流 → 条件筛选 → 候选资产列表
→ 智能尽调（材料解析 → 证据核验 → 尽调报告）→ 智能审批（产品匹配 → 预审 → 补审 → 审批摘要）
```

## 使用步骤

1. 打开百慧系统 → 智能识别 → 文件导入
2. 点击"上传文件"，选择 `名单导入测试_新能源链_20260413.csv`
3. 等待字段校验完成（预期：13条通过，2条需关注）
4. 点击"生成样本"
5. 点击"进入识别"，进入智能推荐流
6. 按照上方表格的预期路径，逐条验证各业务分支
